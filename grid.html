Flex Grid

Our framework a 12 collumn grid. To build the grid you need to define a row, and put many collumn in it.
There is 12 css classes : flex-1 to flex-12. For exemple flex-5 take 5 of 12 collumns of the grid.

The code : 


      <div class="row">
        <div class="flex-2" style="background-color: lightblue;"> size 2 </div>
        <div class="flex-3" style="background-color: lightblue;"> size 3 </div>
        <div class="flex-7" style="background-color: lightblue;"> size 7 </div>
      </div>


Display the following elements :
size 2
size 3
size 7

Automatic flex

Our framework also implement an automated grid. To use it, you need to define a row and put many collumns in it.
The css class flex-auto will grow our shrink to fill the row's width.
If you put 2 flex-auto in 1 row, each element will take half of the width. If you place 4 of them, they will take each quarter.

The code : 


      <div class="row">
        <div class="flex-auto" style="background-color: lightblue;"> a third </div>
        <div class="flex-auto" style="background-color: lightblue;"> a third </div>
        <div class="flex-auto" style="background-color: lightblue;"> a third </div>
      </div>


Display the following elements :
a third
a third
a third
